# CWSP

- Unification of Fastify server (of/for) Frontend/PWA.
- Unification with `endpoint` with `server-v2`.
- Ability to building into `electron` application (include Android).
- Practically same features, as `CWSAndroid` (with/which admin control panel/center, built by `html`, `js` and `css`).
- Ability to run PWA applications (frontend, web).
